星游集结日3月31日:漫威复仇者联盟2、极限对碰、R.B.I.棒球16

游戏 ・ 来源:超好玩游戏网 2016-03-31 19:27

愚人节要来了，想好该怎么打发时间了吗？周四上架的国外新游，倒是不乏有一些恶作剧的创意，如果你并不想在现实中得罪人，那就好好滴在游戏里大搞恶搞一番好啦，来瞧瞧，有什么新游可以让我们愚弄一番~

中二英雄系列

1.《漫威复仇者联盟2 Marvel: Avengers Alliance 2》

上架日期：3月28日

售价：免费

开发商：Marvel Entertainment → 本作为漫威和迪士尼在三年前推出的《漫威复仇者联盟》的手游续作。

玩法类型：动作冒险 → 游戏延续了前作的玩法，为3V3的回合制，较比前作加入了更多的漫威英雄角色，并对游戏改进了不少，如画面有了质的飞跃，从2D变成了3D，对英雄的各类技能有了更炫酷的展现。你在这里将联合复仇者联盟、银河护卫队、神盾局中的各种超级英雄，来对抗奥创、斯特拉克男爵等反派，收集和培养英雄将成为玩法的重中之重。

2.《好骑士的故事 Good Knight Story》

上架日期：3月30日

开发商：Turbo Chilli Pty Ltd

玩法类型：解谜三消 → 游戏的卡通画面十分有趣，不要小看它是款三消游戏，完成消除你会帮助这位“好骑士”在地牢里进行冒险，不断发动技能打败偶遇的怪物，当然不止杀怪那么简单，游戏还加入了幽默的解谜剧情，到底会发生什么事呢？游戏支持中文，一切就等你用消消消的方式去揭晓。?

小清新益智系列

3.《Flail Rider》

开发商：Bulkypix → 游戏在去年就已经先行登陆了Winodws以及Mac等平台，是一款类似推土机类的游戏作品，或许会让你想起诺基亚时代的《疯狂推土机》等小游戏。

玩法类型：休闲益智 → 本作游戏目的就是在场景中大搞破坏，与愚人节多么应景。不过你驾驶的不是推土机，而是拖拽着流星锤的小车，在缤纷的3D几何风格世界里横冲直撞，你将城市肆意碾压破坏得越彻底，得分就越高，只需点屏控制上下左右即可，注意破坏时屏幕也会很晃眼哦，并且要避免你的小车碰壁。

4.《色彩挑战 Kolor Challenge》

上架日期：3月29日

开发商：Thien Dinh

玩法类型：休闲益智 → 如果你不想发泄，那也可以“自虐”一番，本作就是这么一款虐游，你要在恰当的时机发射一个带颜色的小球，并让它安全通过上方的层层障碍到达终点，难就难在如何越过障碍，这些障碍会以五颜六色、奇形怪状的墙壁出现，小球碰到同色的墙壁可通过，异色就会被粉碎，而你就要不断移动旋转的这些障碍，务必让小球能够快速通过。

体育风云系列

5.《R.B.I.棒球16 R.B.I. Baseball 16》

上架日期：3月29日

售价：30元

开发商：MLB.com → 《R.B.I.棒球》系列重启于2014年，是美职棒大联盟的官方授权游戏系列，其游戏一直深得mlb棒球迷的喜爱。

玩法类型：体育竞技 → 游戏里添加了更丰富的模拟功能在赛季模式中，并允许你直接跳到季后赛来玩。在防守、击球等系统较比前作们都有所改进，投球的AI更为真实，美职棒大联盟的30个球场以升级换代的视觉效果出场，画质高清逼真。游戏提供了1100名以上的 MLB 大联盟球员，不用想啦，当然是要你组建自己的球队，开打比赛啦。

6.《极限对碰 Extreme Gear: Demolition Arena》

开发商：Zero Games Studios → 开发商从经典游戏《demolition derby》上获得了灵感，才有了这款类似碰碰车的赛车游戏

玩法类型：赛车竞技→ 愚人节来了，老外一定是和车有仇，这不又有一款撞车游戏，但是这个无疑更凶残，要你去撞别的车，一直撞到车辆爆炸为止。你将驾驶自己的车辆与其他车辆对决，碰撞是你唯一的攻防方式。当然，这可不是莽撞，还要在碰撞中重创对手车的同时尽量降低自身车辆所受到的伤害，这样的技术需要你在游戏里一次次的尝试来积累才行，此外就是升级加强车辆性能了。游戏拥有多个不同的地图，赛车时你还要注意地形对自己的影响，别一不小心被撞到了海里去了。

像素来搞怪系列

7.《锤子炸弹 Hammer Bomb》

开发商：Crescent Moon Games

玩法类型：地牢冒险 → 每周必然少不了的像素游戏又来了，这款游戏画面看似渣得很，但很有意思，这里的迷宫都是随机生成的，你永远不知道前方会出现什么样的环境和怪物，非常考验你的反应能力。你需要快速的躲避前方会出现的障碍物，灵活使用众多道具会来对付怪物，弓箭、锤子、手枪都应有尽有，有时还可能遇到宝箱，这场地牢大暴走注定不平凡。

8.《监狱混战 Prison Run and Gun》

上架日期：3月30日

售价：12元

开发商：Adrian Kumorowski → 本作是之前PC游戏《Hot Guns》的续作，然而前作并没有在移动平台推出。

玩法类型：动作冒险 → 游戏和PC前作相比变化不大，只是新加入了负重设置，所以你在战斗时要考虑武器重量所带来的问题，如移动速度、跳跃高度等都会受此影响，如此本作可不是轻松就能无脑过关的。除了蹦蹦跳跳和打枪之外，选择正确的枪支应对场合也是很重要的。所以你在逃狱时别以为扛着个大枪就能安然无事，一旦不能灵活躲过陷阱和炮火，也会当场丧命。
